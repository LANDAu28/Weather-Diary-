# Weather-Diary-

Пирожков Даниил Олегович

## Описание
Приложение для ведения личного дневника погоды. Позволяет фиксировать ежедневные метеоусловия, сохранять их в локальную базу данных (JSON) и находить нужные записи через систему фильтров.

## Инструкция по запуску
1. Установите зависимости:
   ```bash
   pip install customtkinter
   ```
2. Запустите скрипт:
   ```bash
   python main.py
   ```

## Возможности (Примеры тестов)
1. **Добавление:** Введите `15.05.2024`, `+25`, `Солнечно`, отметьте галочку и нажмите "Добавить".
2. **Валидация:** Попробуйте ввести в температуру буквы — программа выдаст ошибку.
3. **Фильтрация:** Введите в поле фильтра по температуре `20` — в таблице останутся только записи с температурой 20°C и выше.
